Sentiment-Based Aspect Extractor

Discover core aspects of each review related to the emotional tone that underlies it, along with a scoring system that represents that tone.
Overview

Based on the emotions customers express in their reviews, this algorithm extracts identifies aspects from reviews and labels them as positive, negative, or neutral. Additionally, the degree of the emotion's intensity is graded.

How it works

This algorithm takes the extra step of transforming the review if it is unable to extract necessary information to output a concrete result. The transformation is done by using language models. The transformation mainly falls under three categories: summarization, paraphrasing and custom transformation. The system transforms the review with one these models if it fails to extract aspect from that review. If is still fails to find any aspect from it then it attempt to find aspect again with a different model. The models used for these transformation and the sequence of which model should the system use if the aspect mining fails is fully customizable. The users can even provide their own lanugage models to include for the transformations.
